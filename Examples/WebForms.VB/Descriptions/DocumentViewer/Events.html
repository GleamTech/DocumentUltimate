DocumentViewer raises several client-side events:

  • ClientLoad - raised when the viewer has been loaded.
  • ClientError - raised when an error occurs.
  • ClientDocumentLoad - raised when a document has been loaded.
  • ClientPageChange - raised when viewed page is changed.
  • ClientPageComplete - raised when a page had finished rendering.
  • ClientPrint - raised when Print button is clicked.
  • ClientDownload - raised when Download button is clicked.
  • ClientDownloadAsPdf - raised when DownloadAsPdf button is clicked.
  • ClientPrintStart - raised when printing is started.
  • ClientPrintProgress - raised when print progress is changed.

The value of an event property should be a valid JavaScript function name which is accessible on the host page. Function names should be specified without parentheses like "FunctionName" or "Namespace.FunctionName".
